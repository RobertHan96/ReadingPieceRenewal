# ReadingPiece
> 메이커스 WANT팀의 iOS 프로젝트입니다. 
+ 리딩 피스는 꾸준한 목표 달성과 보상으로 독서 습관을 길러줍니다. 
+ 매일 조금씩 책을 읽고 기록을 남겨보세요. 목표에 가까워질수록 맛있는 케이크가 채워집니다. 
+ 커뮤니티를 통해 책에 대한 나의 생각도 공유할 수 있습니다.
